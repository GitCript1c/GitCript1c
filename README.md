## 🧐 About Me

I'm a **2nd-year Software Engineering student** at Bahria University, Islamabad, with hands-on experience building full-stack systems from scratch. I don't just study concepts — I apply them in real projects, like an **event-driven traffic monitoring system** that uses async architecture, WebSockets, React dashboards, and 7 professional design patterns.

I'm comfortable across the entire development stack:
- 🖥️ **Frontend** — React, Tailwind CSS, real-time UI with WebSockets
- ⚙️ **Backend** — Python, FastAPI, Node.js, async/event-driven systems
- 🗄️ **Databases** — MySQL, MongoDB
- 🤖 **AI/ML** — scikit-learn, Pandas, NumPy

What sets me apart is my focus on **writing clean, maintainable code** with proper architecture — not just making things work, but making them work *well*. I take design patterns, code quality, and system scalability seriously even at this early stage of my career.

I'm actively looking for **internship or junior developer opportunities** where I can contribute, grow, and work alongside experienced engineers.
